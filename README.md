This repository contains source code and data relating to the recent publication *"symmetry-protected topological order in a 100-site spin chain on a digital quantum computer"* https://arxiv.org/abs/2603.06325.

## Installation

```sh
git clone https://github.com/stfc/spt-order.git
conda create -n ENV-NAME python=3.12
conda activate ENV-NAME
pip install -e .
```
